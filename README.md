# 🚀 NEXUS ALPHA OSS

Open Source Institutional AI Trading Terminal built for developers, traders, and creators looking to build a custom crypto trading platform, AI assistant, or fintech dashboard.

## ✨ Features
* **Institutional UI**: Responsive, modern dark-themed multi-panel grid.
* **Charts**: Native implementation of TradingView Lightweight Charts with sync capabilities.
* **Live Feed**: Web-polling connection to live Binance Market Data.
* **Indicators Core**: Built-in math processors for indicators like RSI and volatility metrics.
* **AI Signal Panel**: Dedicated zone for processing structured model directions, entries, targets, and invalidations.
* **Risk Widget**: Interactive allocation tool with unit sizing calculators.
